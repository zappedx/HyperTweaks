# HyperTweaks
HyperOS Optimierungen für nicht gerootete Geräte.

[Englisch](https://github.com/zaaaappp/HyperTweaks) - Deutsch

</div>

# Über
HyperTweaks ist ein extrem einfaches ADB basiertes Batch Skript,
das verschiedene Systemeinstellungen optimiert.

> [!CAUTION]
> - Nur auf HyperOS 3 getestet.
> - Stelle sicher, dass du weißt, wie du über die Recovery zurücksetzen kannst.
> - Ich übernehme keine Verantwortung für Bootloops.

# Anleitung
> [!WARNING]
> Stelle sicher, dass USB Debugging aktiviert ist, sonst treten Fehler auf.

Klicke auf den grünen Button "Code" und dann auf "Download ZIP". Lade anschließend [ADB](https://dl.google.com/android/repository/platform-tools-latest-windows.zip) herunter und kopiere folgende Dateien: adb.exe, AdbWinApi.dll, AdbWinUsbApi.dll in die heruntergeladene ZIP-Datei. Danach starte die Datei HyperTweaks.cmd, um dein Handy zu optimieren.

# Was es im Detail macht
- Entfernt verschiedene Xiaomi-Bloatware, bestätigt durch mehrere Quellen
- Aktiviert einen besseren DNS (dns.adguard-dns.com)
- Erhöht oder senkt optional das GPU/CPU Level für bessere Animationen
- Deaktiviert HW Overlay und aktiviert Hardwarebeschleunigung für Tethering

Da kein Root vorhanden ist, kann HyperOS nur begrenzt optimiert werden.
